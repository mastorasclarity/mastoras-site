# Bing Webmaster Tools Setup Guide

## Why Bing Webmaster Tools?

While Google dominates search, Bing powers search for hundreds of millions of users and is the default search engine for many enterprise environments. Bing Webmaster Tools provides:

- **Index Status** - Crawling and indexing information
- **Search Performance** - Keywords, impressions, clicks
- **AI Integration** - Bing AI/Copilot optimization insights
- **Technical Issues** - Crawl errors and fixes
- **Content Submission** - Direct content submission to Bing

## Step-by-Step Setup

### 1. Create Microsoft Account
1. Go to [bing.com/webmasters](https://www.bing.com/webmasters)
2. Sign in with Microsoft account (or create one)
3. Click "Add a site"

### 2. Add Your Property
- **Site URL:** `https://mastoras.netlify.app`
- Click "Add"

### 3. Verify Ownership
Choose "HTML meta tag" method:
1. Copy the meta tag provided
2. Add it to your `index.html` head section
3. Click "Verify"

### 4. Submit Sitemap
1. Go to "Sitemaps" section
2. Enter: `https://mastoras.netlify.app/sitemap.xml`
3. Click "Submit"

### 5. Configure Geotargeting
1. Go to "Site Settings"
2. Set "Geographic target" to "United Kingdom"
3. Set "Preferred domain" to "https://mastoras.netlify.app"
